Path Factorization Playground

Edit the individual end-to-end path values. The tool shows when they can come from shared edge weights—i.e. from two ordinary matrices—and when they are just arbitrary path coefficients.

1. Live matrix editor

A is hidden-by-input. B is output-by-hidden. The product is B·A.

path(i → h → o) = A[h,i] × B[o,h]

2. What must be shared?

A path is not allowed to have a private number. It must reuse the same incoming edge and outgoing edge as every other path touching those edges.

Try this

  1. Edit any matrix cell; the path tables update immediately.
  2. Change one path cell by hand, or click Break one path.
  3. If the path table stays factorable, the matrices update too.
  4. Click Make factorable to project each hidden node back to shared edges.
